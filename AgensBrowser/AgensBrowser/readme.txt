* 파일 구성

- agens-browser-web-1.0.jar
- agens-browser.config.yml
- agensbrowser.bat (windows용)
- agensbrowser.sh (linux 용)

* Script 수행

# Linux
1. agensbrowser.sh 권한을 변경한다.
   예) $ chmod +rx agensbrowser.sh
2. agens-brower 파일 압축해제한 곳으로 이동한다.
3. agensbrowser.sh 파일을 실행한다. 
   예) $ ./agensbrowser.sh

# Windows
1. 관리자 권한으로 cmd를 실행한다.
2. agens-brower 파일 압축해제한 곳으로 이동한다.
3. agensbrowser.bat 파일을 실행한다.
   예) C:\> agensbrowser.bat
